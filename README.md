# Simple Calculator Smart Contract

This project is a **simple calculator smart contract** written in **Solidity** and developed using **Remix IDE**.  
It demonstrates basic smart contract concepts such as functions, modifiers, events, and internal logic.

---

## Features

- Addition and subtraction operations
- Multiplication using stored state
- Custom modifier validation
- Events for tracking operations
- Internal functions for logic separation

---

## Technologies Used

- **Solidity** `^0.8.24`
- **Remix IDE**
- **Ethereum Virtual Machine (EVM)**
